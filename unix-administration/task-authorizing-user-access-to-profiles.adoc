= プロファイルへのアクセスを許可します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager では、ロールベースアクセス制御（ RBAC ）に加えて、プロファイルにパスワードを設定して不正アクセスを防止することができます。

.ステップ
. 次のコマンドを入力します。
+
`* SMSAPのクレデンシャルセット-profile-name_profile_[-password_password_]*`
