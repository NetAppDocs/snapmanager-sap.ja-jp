= SnapManager のアップグレード
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
どのバージョンよりも前のバージョンから、最新バージョンのSnapManager for SAPにアップグレードできます。すべての SnapManager ホストを同時にアップグレードすることも、ローリングアップグレードを実行することもできます。これにより、ホストを段階的にホスト単位でアップグレードできます。
