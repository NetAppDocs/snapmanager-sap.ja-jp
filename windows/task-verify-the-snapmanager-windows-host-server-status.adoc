= SnapManager ホスト・サーバのステータスを確認します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
コマンドを実行したり、 SnapManager 処理を開始したりするには、サーバが実行されている必要があります。処理を実行する前に、サーバのステータスを確認する必要があります。

.手順
. [サービス]ウィンドウで'SAP用のSnapManager 3.3を選択します
. ステータス列でステータスを確認します。
