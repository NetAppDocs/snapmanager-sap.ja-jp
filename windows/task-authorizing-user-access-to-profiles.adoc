= プロファイルへのアクセスの許可
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager では、プロファイルのパスワードを設定して、不正なアクセスを防止できます。

.ステップ
. 次のコマンドを入力します。
+
SMSAPのクレデンシャル・セット-プロファイル名profile_name [-password password
