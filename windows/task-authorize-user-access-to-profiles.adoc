= プロファイルへのアクセスを許可します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager では、プロファイルのパスワードを設定して、不正なアクセスを防止できます。

.ステップ
. 次のコマンドを入力します。
+
`* SMSAPのクレデンシャルセット-profile-name_profile_[-password_password_]*`
