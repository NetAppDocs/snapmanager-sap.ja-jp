= SnapManager ホストサーバを停止します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapManager の使用が終了したら、必要に応じてサーバを停止できます。

. サーバを停止するには、rootユーザとして次のコマンドを入力します。
+
「smsap_server stop」という2つのサーバがあります
